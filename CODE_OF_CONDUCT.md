# Code of Conduct

## Purpose

Living Atlas is a public collaboration space for building a private-first
knowledge graph system. The project works best when contributors can discuss
security, privacy, architecture, and implementation tradeoffs directly while
remaining respectful and constructive.

## Expected Behavior

Participants are expected to:

- Be respectful of differing experience levels and perspectives.
- Critique ideas, code, and decisions rather than people.
- Keep discussions relevant to the project and its users.
- Prefer clear, evidence-based technical feedback.
- Respect privacy boundaries and avoid sharing private data, secrets, or
  deployment-specific details.
- Accept maintainer decisions once a decision has been made, while preserving
  room for future evidence and proposals.

## Unacceptable Behavior

Unacceptable behavior includes:

- Harassment, threats, insults, or sustained personal attacks.
- Discriminatory language or conduct.
- Sexualized language or unwanted sexual attention.
- Publishing or threatening to publish private information.
- Sharing another person's data, credentials, deployment details, logs, or graph
  content without authorization.
- Deliberately disrupting issues, pull requests, reviews, or project spaces.
- Encouraging others to violate this code of conduct.

## Scope

This code applies in project-managed spaces, including issues, pull requests,
discussions, documentation, code review, release activity, and community
channels connected to Living Atlas. It also applies when someone represents the
project in a public context.

## Reporting and Enforcement

Report conduct concerns to maintainers through the repository's published
private contact path when one is available. If no private path is available,
open a minimal public issue requesting a private maintainer contact without
including sensitive details.

Maintainers may remove content, close or lock discussions, reject
contributions, restrict participation, or take other reasonable action to
protect the project and its participants. Enforcement decisions should be
proportionate to the behavior, the surrounding context, and the safety of the
community.
